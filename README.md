
# ai-for-bharat-submission

This repository contains the documentation for **BharatSense**, an offline-first, multilingual AI copilot designed to help citizens and frontline workers access government schemes with low connectivity and low digital literacy.

## Contents
- requirements.md – Problem, objectives, features, constraints
- design.md – System architecture and technical design

## Note
This is a documentation-focused submission as per hackathon guidelines. No code implementation is included.
